Quad4R useful functions (pyfe3d.quad4r_utils)¶

pyfe3d.quad4r_utils.quad4r_coord(quad: Quad4R, ncoords: ndarray)¶

Determine the coordinate system of a Quad4R element

The element coordinate system is determined identically what is explained in Nastran’s quick reference guide for the CQUAD4 element, as illustrated below.

Parameters:
quadQuad4R object

The quadilateral element.

ncoordsnp.ndarray object

An array with shape \(N, 3\) for a system with \(N\) nodes.
